# GeaSlider

## どんなやつ？
Markdownからスライドを作成するCLI。（全然未完成）
一般的なMarkdown記法に、ZennのMarkdown記法をリスペクトし、
スライド作成に活かせるように調整したCLI（として開発する予定）

## 柔軟なカスタマイズ性
デザイン設定は、デフォルトで用意されているCSSを書き換えれば、
自分の好きなデザインに簡単に変更可能（なように設計する予定）

## 高速な処理
Rustで実装しているため高速に動作（するように実装する予定）
